O que é CSS?

Para entender de maneira simples, pense no HTML como a estrutura básica de
uma página da web,como os tijolos de uma casa. Enquanto isso, o CSS é como a
pintura,os móveise a decoraçãoque você adiciona para tornar a casa bonita e
aconchegante.

Em resumo, o CSS é usado para estilizar e embelezar páginas da web, separando a
apresentação visual do conteúdo estruturado definido pelo HTML.
